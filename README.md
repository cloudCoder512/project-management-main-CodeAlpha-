# 🚀 Project Manager App

A modern full-stack Project Management Web Application built to manage tasks, teams, and workflows efficiently. This app allows users to create projects, assign tasks, track progress, and collaborate in real-time.

---

## 📌 Features

- ✅ User Authentication (Login / Signup)
- 📁 Create & Manage Projects
- 📝 Task Assignment & Tracking
- 📊 Dashboard Overview
- 👥 Team Collaboration
- 🔔 Notifications (optional if implemented)
- 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Vite
- Material UI / Mantine
- Axios

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 🔹 Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/project-manager.git
cd project-manager
```
### Step 2: Install Dependencies
```bash
npm install
cd frontend
npm run dev
```
